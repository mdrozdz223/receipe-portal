# Recipe Portal 🍳

A full-stack web application for discovering, sharing, and managing cooking recipes. Built with a modern tech stack featuring a Spring Boot backend and a React frontend.

## 🚀 Features
* Browse a rich collection of recipes with ingredients and step-by-step instructions.
* Add, edit, and manage your own culinary creations.
* Search and filter functionality to easily find specific meals.
* Responsive, user-friendly Single Page Application (SPA) design.

## 🛠️ Tech Stack
* **Frontend:** React, HTML5, CSS3
* **Backend:** Java, Spring Boot, REST API
* **Database:** PostgreSQL

## 🔧 Getting Started

To run this project locally, you will need Java and Node.js installed on your machine.

### Backend Setup
1. Navigate to the backend directory.
2. Build and run the Spring Boot application:
```bash
   ./mvnw spring-boot:run
   ```

### Frontend Setup
1. Navigate to the frontend directory.
2. Install dependencies and start the development server:
```bash
   npm install
   npm start
   ```
